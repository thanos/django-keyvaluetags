A simple key=value tagging system a la AWS.

Documentation
-------------

The full documentation is at https://django-keyvaluetags.readthedocs.org.

Quickstart
----------

Install django-keyvaluetags::

    pip install django-keyvaluetags

Then use it in a project::

    import keyvaluetags
